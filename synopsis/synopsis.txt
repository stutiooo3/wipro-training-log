// Understanding Promises and Asynchronous JavaScript

1. A Promise is used to handle operations that take time, such as fetching data from a server. It has three states:
* Pending - initial state
* Resolved (Fulfilled) - operation completed successfully
* Rejected - operation failed

2. In the 'getData()' function, a Promise is created using 'new Promise()'. It takes two parameters:
* resolve - called when the task is successful
* reject - called when an error occurs

3. setTimeout()
A built-in function used to delay execution of code.
Runs a function once after a specified time (in milliseconds).
Commonly used to simulate asynchronous tasks (e.g., API calls).

4. Handling Promises
.then()- Executes when the Promise is resolved , Receives the result (data)
.catch()- Executes when the Promise is rejected, Handles errors

5. setInterval()
Executes a function repeatedly at fixed time intervals and continues running until stopped manually.

6. Non-blocking Execution
JavaScript executes asynchronous code without stopping the rest of the program.
Promises help manage async operations in a cleaner and more readable way than callbacks.


 Example: Promise using setTimeout

function getData() {
    return new Promise((resolve, reject) => {
        setTimeout(() => {
            resolve("Data received");
        }, 2000);
    });
}

getData()
    .then(result => console.log(result))
    .catch(error => console.error(error));


* Function returns a Promise
* setTimeout simulates delay (2 seconds)
* resolve() sends success data
* .then() handles success
* .catch() handles errors (not used here since no reject)

7. Timers in JavaScript

* setTimeout() - runs once after delay
* setInterval() - runs repeatedly after fixed interval

Example:

setTimeout(() => console.log("Runs once"), 2000);

setInterval(() => console.log("Repeats every 3 sec"), 3000);

8. Fetch API (Getting Data from Server)

The Fetch API is used to request data from an external source (API).

* fetch() returns a Promise
* Data must be converted into JSON format using .json()

9. Processing API Data

After fetching data, we usually:
1. Convert response to JSON
2. Extract required data
3. Filter specific items
4. Modify data
5. Display results

Example: Fetch + Filter + Map

fetch("https://jsonplaceholder.typicode.com/todos")
    .then(res => res.json())
    .then(data => data.filter(item => item.id % 2 === 0))
    .then(data => data.map(item => ({
        ...item,
        title: "Updated " + item.title
    })))
    .then(data => console.log(data))
    .catch(err => console.error("Error:", err));

10. Flowchart

fetch() - get data 
convert to JSON
filter required items
transform data (map)
display output
handle errors

10. Key Methods Used

* .then()- handles successful result
* .catch()-handles errors
* .filter()- selects specific data
* .map()-modifies or transforms data
